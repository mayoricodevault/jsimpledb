031/**
032 * A JSimpleDB {@link Database} transaction.
033 *
034 * <p>
035 * Methods in this class can be divided into the following categories:
036 * </p>
037 *
038 * <p>
039 * <b>Transaction Meta-Data</b>
040 * <ul>
041 *  <li>{@link #getDatabase getDatabase()} - Get the associated {@link Database}
042 *  <li>{@link #getSchema getSchema()} - Get the database {@link Schema}, as seen by this transaction
043 *  <li>{@link #getSchemaVersion() getSchemaVersion()} - Get the {@link SchemaVersion} that will be used by this transaction
044 * </ul>
045 * </p>
046 *
047 * <p>
048 * <b>Transaction Lifecycle</b>
049 * <ul>
050 *  <li>{@link #commit commit()} - Commit transaction
051 *  <li>{@link #rollback rollback()} - Roll back transaction
052 *  <li>{@link #isValid isValid()} - Test transaction validity
053 *  <li>{@link #setTimeout setTimeout()} - Set transaction timeout
054 *  <li>{@link #setReadOnly setReadOnly()} - Set transaction to read-only
055 *  <li>{@link #setRollbackOnly setRollbackOnly()} - Set transaction for rollack only
056 *  <li>{@link #addCallback addCallback()} - Register a {@link Callback} on transaction completion
057 * </ul>
058 * </p>
059 *
060 * <p>
061 * <b>Object Lifecycle</b>
062 * <ul>
063 *  <li>{@link #create(int) create()} - Create a database object
064 *  <li>{@link #delete delete()} - Delete a database object
065 *  <li>{@link #exists exists()} - Test whether a database object exists
066 *  <li>{@link #addCreateListener addCreateListener()} - Register a {@link CreateListener} for notifications about new objects
067 *  <li>{@link #removeCreateListener removeCreateListener()} - Unregister a {@link CreateListener}
068 *  <li>{@link #addDeleteListener addDeleteListener()} - Register a {@link DeleteListener} for notifications about object deletions
069 *  <li>{@link #removeDeleteListener removeDeleteListener()} - Unregister a {@link DeleteListener}
070 * </ul>
071 * </p>
072 *
073 * <p>
074 * <b>Object Versioning</b>
075 * <ul>
076 *  <li>{@link #getSchemaVersion(ObjId) getSchemaVersion()} - Inspect an object's schema version
077 *  <li>{@link #updateSchemaVersion updateSchemaVersion()} - Update an object's schema version to match this transaction
078 *  <li>{@link #addVersionChangeListener addVersionChangeListener()} - Register a {@link VersionChangeListener} for notifications
079 *      about object version changes
080 *  <li>{@link #removeVersionChangeListener removeVersionChangeListener()} - Unregister a {@link VersionChangeListener}
081 * </ul>
082 *
083 * <p>
084 * <b>Object and Field Access</b>
085 * <ul>
086 *  <li>{@link #getAll getAll()} - Get all objects of a specific type
087 *  <li>{@link #readSimpleField readSimpleField()} - Read the value of a {@link SimpleField} in an object
088 *  <li>{@link #writeSimpleField writeSimpleField()} - Write the value of a {@link SimpleField} in an object
089 *  <li>{@link #readSetField readSetField()} - Access a {@link SetField} in an object as a {@link NavigableSet}
090 *  <li>{@link #readListField readListField()} - Access a {@link ListField} in an object as a {@link List}
091 *  <li>{@link #readMapField readMapField()} - Access a {@link MapField} in an object as a {@link NavigableMap}
092 *  <li>{@link #getKey getKey(ObjId)} - Get the {@link org.jsimpledb.kv.KVDatabase} key corresponding to an object
093 *  <li>{@link #getKey getKey(ObjId, int)} - Get the {@link org.jsimpledb.kv.KVDatabase}
094 *      key corresponding to a field in an object
095 * </ul>
096 * </p>
097 *
098 * <p>
099 * <b>Field Change Notifications</b>
100 * <ul>
101 *  <li>{@link #addSimpleFieldChangeListener addSimpleFieldChangeListener()} - Register a {@link SimpleFieldChangeListener} for
102 *      notifications of changes in a {@link SimpleField}, as seen through a path of object references
103 *  <li>{@link #addSetFieldChangeListener addSetFieldChangeListener()} - Register a {@link SetFieldChangeListener} for
104 *      notifications of changes in a {@link SetField}, as seen through a path of object references
105 *  <li>{@link #addListFieldChangeListener addListFieldChangeListener()} - Register a {@link ListFieldChangeListener} for
106 *      notifications of changes in a {@link ListField}, as seen through a path of object references
107 *  <li>{@link #addMapFieldChangeListener addMapFieldChangeListener()} - Register a {@link MapFieldChangeListener} for
108 *      notifications of changes in a {@link MapField}, as seen through a path of object references
109 *  <li>{@link #removeSimpleFieldChangeListener removeSimpleFieldChangeListener()} - Unregister a previously registered
110 *      {@link SimpleFieldChangeListener}
111 *  <li>{@link #removeSetFieldChangeListener removeSetFieldChangeListener()} - Unregister a previously registered
112 *      {@link SetFieldChangeListener}
113 *  <li>{@link #removeListFieldChangeListener removeListFieldChangeListener()} - Unregister a previously registered
114 *      {@link ListFieldChangeListener}
115 *  <li>{@link #removeMapFieldChangeListener removeMapFieldChangeListener()} - Unregister a previously registered
116 *      {@link MapFieldChangeListener}
117 * </ul>
118 * </p>
119 *
120 * <p>
121 * <b>Reference Path Queries</b>
122 * <ul>
123 *  <li>{@link #invertReferencePath invertReferencePath()} - Find all objects that refer to any element in a given set
124 *      of objects through a specified reference path
125 *  </li>
126 * </ul>
127 * </p>
128 *
129 * <p>
130 * <b>Index Queries</b>
131 * <ul>
132 *  <li>{@link #querySimpleField querySimpleField()} - Query the index associated with a {@link SimpleField}
133 *      to identify all values and all objects having those values
134 *  <li>{@link #querySetField querySetField()} - Query the index associated with a {@link SetField}
135 *      to identify all set elements and all objects having those elements in the set
136 *  <li>{@link #queryListField queryListField()} - Query the index associated with a {@link ListField}
137 *      to identify all list elements and all objects having those elements in the list
138 *  <li>{@link #queryListFieldEntries queryListFieldEntries()} - Query the index associated with a {@link ListField}
139 *      to identify all list elements and all objects with those elements in the list, as well as the associated list indicies
140 *  <li>{@link #queryMapFieldKey queryMapFieldKey()} - Query the index associated with a {@link MapField}
141 *      to identify all map keys and all objects having those keys in the map
142 *  <li>{@link #queryMapFieldKeyEntries queryMapFieldKeyEntries()} - Query the index associated with a {@link MapField}
143 *      to identify all map keys and all objects having those keys in the map, as well as the associated map values
144 *  <li>{@link #queryMapFieldValue queryMapFieldValue()} - Query the index associated with a {@link MapField}
145 *      to identify all map values and all objects having those values in the map
146 *  <li>{@link #queryMapFieldValueEntries queryMapFieldValueEntries()} - Query the index associated with a {@link MapField}
147 *      to identify all map values and all objects having those values in the map, as well as the associated map keys
148 * </ul>
149 * </p>
150 *
151 * <p>
152 * All methods returning a set of values return a {@link NavigableSet}.
153 * The {@link NavigableSets} utility class provides methods for the efficient {@link NavigableSets#intersection intersection},
154 * {@link NavigableSets#union union}, {@link NavigableSets#difference difference}, and
155 * {@link NavigableSets#symmetricDifference symmetric difference} of {@link NavigableSet}s containing the same element type,
156 * thereby providing the equivalent of traditional database joins.
157 * </p>
158 */

228    /**
229     * Commit this transaction.
230     *
231     * @throws StaleTransactionException if this transaction is no longer usable
232     * @throws RetryTransactionException from {@link KVTransaction#commit KVTransaction.commit()}
233     * @throws RollbackOnlyTransactionException if this instance has been {@linkplain #setRollbackOnly marked} rollback only;
234     *  this instance will be automatically rolled back
235     */
236    public synchronized void commit() {
237
238        // Sanity check
239        if (this.stale)
240            throw new StaleTransactionException(this);
241
242        // Rollback only?
243        if (this.rollbackOnly) {
244            this.log.info("commit() invoked on transaction " + this + " marked rollback-only, rolling back");
245            this.rollback();
246            throw new RollbackOnlyTransactionException(this);
247        }
248        this.stale = true;
249
250        // Do before completion callbacks
251        this.log.debug("commit() invoked on" + (this.readOnly ? " read-only" : "") + " transaction " + this);
252        Callback failedCallback = null;
253        try {
254            for (Callback callback : this.callbacks) {
255                failedCallback = callback;
256                callback.beforeCommit(this.readOnly);
257            }
258            failedCallback = null;
259        } finally {
260            if (failedCallback != null) {
261                this.log.warn("error invoking beforeCommit() method on transaction "
262                  + this + " callback " + failedCallback + ", rolling back");
263            }
264            this.triggerBeforeCompletion();
265            if (failedCallback != null) {
266                try {
267                    this.kvt.rollback();
268                } finally {
269                    this.triggerAfterCompletion(false);
270                }
271            }
272        }
273
274        // Commit KVTransaction and trigger after completion callbacks
275        failedCallback = null;
276        try {
277            this.kvt.commit();
278            for (Callback callback : this.callbacks) {
279                failedCallback = callback;
280                callback.afterCommit();
281            }
282            failedCallback = null;
283        } finally {
284            if (failedCallback != null)
285                this.log.warn("error invoking afterCommit() method on transaction " + this + " callback " + failedCallback);
286            this.triggerAfterCompletion(true);
287        }
288    }

290    /**
291     * Roll back this transaction.
292     *
293     * @throws StaleTransactionException if this transaction is no longer usable
294     */
295    public synchronized void rollback() {
296
297        // Sanity check
298        if (this.stale)
299            throw new StaleTransactionException(this);
300        this.stale = true;
301        this.log.debug("rollback() invoked on" + (this.readOnly ? " read-only" : "") + " transaction " + this);
302
303        // Do before completion callbacks
304        this.triggerBeforeCompletion();
305
306        // Roll back KVTransaction and trigger after completion callbacks
307        try {
308            this.kvt.rollback();
309        } finally {
310            this.triggerAfterCompletion(false);
311        }
312    }

314    private /*synchronized*/ void triggerBeforeCompletion() {
315        for (Callback callback : this.callbacks) {
316            try {
317                callback.beforeCompletion();
318            } catch (Throwable t) {
319                this.log.error("error invoking beforeCompletion() method on transaction callback " + callback, t);
320            }
321        }
322    }
323
324    private /*synchronized*/ void triggerAfterCompletion(boolean committed) {
325        for (Callback callback : this.callbacks) {
326            try {
327                callback.afterCompletion(committed);
328            } catch (Throwable t) {
329                this.log.error("error invoking afterCompletion() method on transaction callback " + callback, t);
330            }
331        }
332    }

418    /**
419     * Create a new object with the given object ID, if it doesn't already exist. If it does exist, nothing happens.
420     *
421     * <p>
422     * If the object doesn't already exist, all fields are set to their default values and the object's
423     * schema version is set to {@linkplain #getSchemaVersion the version associated with this transaction}.
424     * </p>
425     *
426     * @param id object ID
427     * @return true if the object did not exist and was created, false if the object already existed
428     * @throws IllegalArgumentException if the storage ID associated with {@code id}
429     *  does not correspond to a known object type in this transaction's schema
430     * @throws IllegalArgumentException if {@code id} is null

434    public synchronized boolean create(ObjId id) {
435
436        // Sanity check
437        if (id == null)
438            throw new IllegalArgumentException("null id");
439        if (this.stale)
440            throw new StaleTransactionException(this);
441        if (this.readOnly)
442            throw new ReadOnlyTransactionException(this);
443
444        // Does object already exist?
445        if (this.exists(id))
446            return false;
447
448        // Initialize object
449        this.initialize(id, this.version.getSchemaItem(id.getStorageId(), ObjType.class));
450
451        // Done
452        return true;
453    }

455    /**
456     * Create a new object with a randomly assigned object ID and having the given type.
457     *
458     * <p>
459     * All fields will be set to their default values.
460     * The object's schema version will be set to {@linkplain #getSchemaVersion the version associated with this transaction}.
461     * </p>
462     *
463     * @param storageId object type storage ID
464     * @return object id of newly created object
465     * @throws IllegalArgumentException if {@code storageId} does not correspond to a known object type in this transaction's schema
466     * @throws ReadOnlyTransactionException if this transaction has been {@linkplain #setReadOnly set read-only}
467     * @throws StaleTransactionException if this transaction is no longer usable
468     */
469    public synchronized ObjId create(int storageId) {
470
471        // Sanity check
472        if (this.stale)
473            throw new StaleTransactionException(this);
474        if (this.readOnly)
475            throw new ReadOnlyTransactionException(this);
476        final ObjType objType = this.version.getSchemaItem(storageId, ObjType.class);
477
478        // Create a new, unique key
479        ObjId id;
480        final ByteWriter keyWriter = new ByteWriter();
481        int attempts = 0;
482        while (true) {
483            id = new ObjId(objType.storageId);
484            id.writeTo(keyWriter);
485            if (this.kvt.get(keyWriter.getBytes()) == null)
486                break;
487            if (++attempts == MAX_UNIQUE_KEY_ATTEMPTS) {
488                throw new DatabaseException("could not find a new, unused object ID after "
489                  + attempts + " attempts; is our source of randomness truly random?");
490            }
491            keyWriter.reset(0);
492        }
493
494        // Initialize object
495        this.initialize(id, objType);
496
497        // Done
498        return id;
499    }

501    private synchronized void initialize(ObjId id, ObjType objType) {
502
503        // Write object meta-data
504        ObjInfo.write(this, id, objType.version.versionNumber, false);
505
506        // Write simple field index entries
507        for (SimpleField<?> field : objType.simpleFields.values()) {
508            if (field.indexed)
509                this.kvt.put(field.buildIndexKey(id, null), ByteUtil.EMPTY);
510        }
511
512        // Notify listeners
513        for (CreateListener listener : this.createListeners.toArray(new CreateListener[this.createListeners.size()]))
514            listener.onCreate(this, id);
515    }

517    /**
518     * Delete an object. Does nothing if object does not exist (e.g., has already been deleted).
519     *
520     * <p>
521     * This method does <i>not</i> change the object's schema version if it has a different version that this transaction.
522     * </p>
523     *
524     * @param id object ID of the object to delete
525     * @return true if object was found and deleted, false if object was not found
526     * @throws ReferencedObjectException if the object is referenced by some other object
527     *  through a reference field set to {@link DeleteAction#EXCEPTION}
528     * @throws IllegalArgumentException if {@code id} is null
529     * @throws ReadOnlyTransactionException if this transaction has been {@linkplain #setReadOnly set read-only}
530     * @throws StaleTransactionException if this transaction is no longer usable
531     */
532    public synchronized boolean delete(ObjId id) {
533
534        // Sanity check
535        if (id == null)
536            throw new IllegalArgumentException("null id");
537        if (this.stale)
538            throw new StaleTransactionException(this);
539        if (this.readOnly)
540            throw new ReadOnlyTransactionException(this);
541
542        // Handle recurive DeleteAction.DELETE without hogging Java stack
543        final ArrayList<ObjId> deletables = new ArrayList<>(1);
544        deletables.add(id);
545        boolean found = false;
546        while (true) {
547            final int size = deletables.size();
548            if (size == 0)
549                break;
550            id = deletables.remove(size - 1);
551            found |= this.doDelete(id, deletables);
552        }
553        return found;
554    }
555
556    private synchronized boolean doDelete(final ObjId id, List<ObjId> deletables) {
557
558        // Loop here to handle any mutations within delete notification listener callbacks
559        ObjInfo info;
560        while (true) {
561
562            // Get object info
563            try {
564                info = this.getObjectInfo(id, false);
565            } catch (DeletedObjectException e) {                    // possibly due to a cycle of DeleteAction.DELETE references
566                return false;
567            }
568
569            // Determine if any EXCEPTION reference fields refer to the object (from some other object); if so, throw exception
570            for (ReferenceFieldStorageInfo fieldInfo : this.schema.exceptionReferenceFieldStorageInfos) {
571                final NavigableSet<ObjId> referrers = this.findReferences(fieldInfo, id);
572                if (referrers == null)
573                    continue;
574                for (ObjId referrer : referrers) {
575                    if (!referrer.equals(id))
576                        throw new ReferencedObjectException(id, referrer, fieldInfo.storageId);
577                }
578            }
579
580            // Need to issue delete notifications?
581            if (info.isDeleteNotified() || this.deleteListeners.isEmpty())
582                break;
583
584            // Issue delete notifications and retry
585            ObjInfo.write(this, id, info.getVersionNumber(), true);
586            for (DeleteListener listener : this.deleteListeners.toArray(new DeleteListener[this.deleteListeners.size()]))
587                listener.onDelete(this, id);
588        }
589
590        // Delete object's simple field index entries
591        final ObjType type = info.getObjType();
592        for (SimpleField<?> field : type.simpleFields.values()) {
593            if (field.indexed)
594                this.kvt.remove(field.buildIndexKey(id, this.kvt.get(field.buildKey(id))));
595        }
596
597        // Delete object's complex field index entries
598        for (ComplexField<?> field : type.complexFields.values()) {
599            for (SimpleField<?> subField : field.getSubFields()) {
600                if (subField.indexed)
601                    field.removeIndexEntries(this, id, subField);
602            }
603        }
604
605        // Delete object version and all field content
606        final byte[] minKey = info.getId().getBytes();
607        final byte[] maxKey = ByteUtil.getKeyAfterPrefix(minKey);
608        this.kvt.removeRange(minKey, maxKey);
609
610        // Find all references to this object and deal with them
611        for (ReferenceFieldStorageInfo fieldInfo : this.schema.referenceFieldStorageInfos) {
612            if (fieldInfo.onDelete == DeleteAction.EXCEPTION)           // we already checked these
613                continue;
614            final NavigableSet<ObjId> referrers = this.findReferences(fieldInfo, id);
615            if (referrers == null)
616                continue;
617            switch (fieldInfo.onDelete) {
618            case NOTHING:
619                break;
620            case UNREFERENCE:
621            {
622                final int storageId = fieldInfo.storageId;
623                if (fieldInfo.isSubField()) {
624                    final ComplexFieldStorageInfo superFieldInfo = this.schema.verifyStorageInfo(
625                      fieldInfo.superFieldStorageId, ComplexFieldStorageInfo.class);
626                    superFieldInfo.unreferenceAll(this, storageId, id);
627                } else {
628                    for (ObjId referrer : referrers)
629                        this.writeSimpleField(referrer, storageId, null);
630                }
631                break;
632            }
633            case DELETE:
634                deletables.addAll(referrers);
635                break;
636            default:
637                throw new RuntimeException("unexpected case");
638            }
639        }
640
641        // Done
642        return true;
643    }

786    /**
787     * Update object to the current schema version.
788     *
789     * @param id object id
790     * @param info original object info
791     * @throws ReadOnlyTransactionException if this transaction has been {@linkplain #setReadOnly set read-only}
792     * @throws InvalidObjectVersionException if the schema version is invalid
793     * @throws IllegalArgumentException if {@code id} is null
794     */
795    private synchronized void updateVersion(ObjId id, ObjInfo info) {
796
797        // Sanity check
798        if (info.getVersionNumber() == this.version.versionNumber)
799            throw new IllegalArgumentException("object already at version " + this.version.versionNumber);
800        if (this.readOnly)
801            throw new ReadOnlyTransactionException(this);
802
803        // Get old and new types
804        final ObjType oldType = info.getObjType();
805        final ObjType newType;
806        try {
807            newType = this.version.getSchemaItem(id.getStorageId(), ObjType.class);
808        } catch (IllegalArgumentException e) {
809            throw new InconsistentDatabaseException("object " + id + " has an unrecognized storage ID", e);
810        }
811
812    //////// Update simple fields and corresponding index entries
813
814        // Gather removed fields' values here for user migration
815        final TreeMap<Integer, Object> oldFieldValues = new TreeMap<>();
816
817        // Get simple field storage IDs (old or new)
818        final TreeSet<Integer> simpleFieldStorageIds = new TreeSet<>();
819        simpleFieldStorageIds.addAll(oldType.simpleFields.keySet());
820        simpleFieldStorageIds.addAll(newType.simpleFields.keySet());
821
822        // Update simple fields
823        for (int storageId : simpleFieldStorageIds) {
824
825            // Get old and new simple fields having this storage ID
826            final SimpleField<?> oldField = oldType.simpleFields.get(storageId);
827            final SimpleField<?> newField = newType.simpleFields.get(storageId);
828
829            // Save old field values for version change notification
830            final byte[] key = SimpleField.buildKey(id, storageId);
831            final byte[] oldValue = this.kvt.get(key);
832            if (oldField != null) {
833                oldFieldValues.put(storageId,
834                  oldField.fieldType.read(new ByteReader(oldValue != null ? oldValue : oldField.fieldType.getDefaultValue())));
835            }
836
837            // Preserve the value if we can, otherwise, discard the old value (if any) leaving new value as new field default
838            boolean skipIndexUpdate = false;
839            if (oldField != null && newField != null && newField.isCompatible(oldField))    // value is compatible so just leave it
840                skipIndexUpdate = newField.indexed == oldField.indexed;                     // index need not change either
841            else if (oldValue != null)
842                this.kvt.remove(key);                                                       // discard old value
843
844            // Remove old index entry
845            if (oldField != null && oldField.indexed && !skipIndexUpdate)
846                this.kvt.remove(oldField.buildIndexKey(id, oldValue));
847
848            // Add new index entry
849            if (newField != null && newField.indexed && !skipIndexUpdate)
850                this.kvt.put(newField.buildIndexKey(id, null), ByteUtil.EMPTY);
851        }
852
853        // Update object version
854        ObjInfo.write(this, id, this.version.versionNumber, info.isDeleteNotified());
855
856    //////// Update complex fields and corresponding index entries
857
858        // Keep track of what to delete when we've done the migration notification
859        final ArrayList<ComplexField<?>> cleanupList = new ArrayList<>();
860
861        // Get complex field storage IDs (old or new)
862        final TreeSet<Integer> complexFieldStorageIds = new TreeSet<>();
863        complexFieldStorageIds.addAll(oldType.complexFields.keySet());
864        complexFieldStorageIds.addAll(newType.complexFields.keySet());
865
866        // Notes:
867        //
868        // - The only "migration" we support is adding/removing sub-field indexes; all other changes are equivalent to delete+add
869        // - New complex fields do not need to be explicitly initialized because their initial state is to have zero KV pairs
870        //
871        for (int storageId : complexFieldStorageIds) {
872
873            // Get old and new complex fields having this storage ID
874            final ComplexField<?> oldField = oldType.complexFields.get(storageId);
875            final ComplexField<?> newField = newType.complexFields.get(storageId);
876
877            // If there is no old field, new field is new and so it is already initialized (empty)
878            if (oldField == null)
879                continue;
880
881            // Save old field's value
882            oldFieldValues.put(storageId, oldField.getValue(this, id));
883
884            // Determine if the fields are compatible and field content can be preserved
885            final boolean compatible = newField != null && newField.isCompatible(oldField);
886
887            // If fields are not compatible, delete old field when done, otherwise check if index(s) should be added/removed
888            if (!compatible)
889                cleanupList.add(oldField);
890            else
891                newField.updateSubFieldIndexes(this, oldField, id);
892        }
893
894        // Notify about version update
895        try {
896            for (VersionChangeListener listener : this.versionChangeListeners)
897                listener.onVersionChange(this, id, info.getVersionNumber(), this.version.versionNumber, oldFieldValues);
898        } finally {
899            for (ComplexField<?> oldField : cleanupList)
900                oldField.deleteContent(this, id);
901        }
902    }

1049    private synchronized void doWriteSimpleField(ObjId id, int storageId, final Object newObj) {
1050
1051        // Get object info
1052        final ObjInfo info = this.getObjectInfo(id, true);
1053        final ObjType type = info.getObjType();
1054
1055        // Find field
1056        final SimpleField<?> field = type.simpleFields.get(storageId);
1057        if (field == null)
1058            throw new UnknownFieldException(type, storageId, "simple field");
1059
1060        // Get old and new values
1061        final byte[] key = field.buildKey(id);
1062        final byte[] oldValue = this.kvt.get(key);
1063        final byte[] newValue = field.encode(newObj);
1064
1065        // Optimization: check if the value has not changed
1066        if (oldValue != null ? newValue != null && Arrays.equals(oldValue, newValue) : newValue == null)
1067            return;
1068
1069        // Update value
1070        if (newValue != null)
1071            this.kvt.put(key, newValue);
1072        else
1073            this.kvt.remove(key);
1074
1075        // Update index
1076        if (field.indexed) {
1077            this.kvt.remove(field.buildIndexKey(id, oldValue));
1078            this.kvt.put(field.buildIndexKey(id, newValue), ByteUtil.EMPTY);
1079        }
1080
1081        // Notify monitors
1082        final Object oldObj = field.fieldType.read(new ByteReader(oldValue != null ? oldValue : field.fieldType.getDefaultValue()));
1083        this.addFieldChangeNotification(new SimpleFieldChangeNotifier(field, id) {
1084            @Override
1085            public void notify(Transaction tx, SimpleFieldChangeListener listener, int[] path, NavigableSet<ObjId> referrers) {
1086                listener.onSimpleFieldChange(tx, this.id, this.storageId, path, referrers, oldObj, newObj);
1087            }
1088        });
1089    }

1262            throw new IllegalArgumentException("storageId <= 0");

1269        id.writeTo(writer);
1270        UnsignedIntEncoder.write(writer, storageId);

1272    }

1321    /**
1322     * Monitor for changes within this transaction of the value of the given field, as seen through a path of references.
1323     *
1324     * <p>
1325     * When the specified field is changed, a listener notification will be delivered for each referring object that
1326     * refers to the object containing the changed field through the specified path of references. Notifications are
1327     * delivered at the end of the mutation operation just prior to returning to the caller. If the listener method
1328     * performs additional mutation(s) which are themselves being listened for, those notifications will also be delivered
1329     * prior to the returning to the original caller. Therefore, care must be taken to avoid change notification dependency
1330     * loops when listeners can themselves mutate fields, to avoid infinite loops.
1331     * </p>
1332     *
1333     * <p>
1334     * A referring object may refer to the changed object through more than one actual path of references matching {@code path};
1335     * if so, it will still appear only once in the {@link NavigableSet} provided to the listener (this is of course required
1336     * by set semantics).
1337     * </p>
1338     *
1339     * <p>
1340     * Although the reference back-tracking algorithm does consolidate multiple paths between the same two objects,
1341     * be careful to avoid an explosion of notifications when objects in the {@code path} have a high degree of fan-in.
1342     * </p>
1343     *
1344     * <p>
1345     * When a {@link ReferenceField} in {@code path} also happens to be the field being changed, then there is ambiguity
1346     * about how the set of referring objects is calculated: does it use the field's value before or after the change?
1347     * This API guarantees that the answer is "after the change"; however, if another listener on the same field is
1348     * invoked before {@code listener} and mutates any reference field(s) in {@code path}, then whether that additional
1349     * change is also be included in the calculation is undefined.
1350     * </p>
1351     *
1352     * <p>
1353     * Therefore, for consistency, avoid changing any {@link ReferenceField} from within a listener callback when that
1354     * field is also in some other listener's reference path, and both listeners are watching the same field.
1355     * </p>
1356     *
1357     * @param storageId storage ID of the field to monitor
1358     * @param path path of reference fields (represented by storage IDs) through which to monitor field
1359     * @param listener callback for notifications on changes in value
1360     * @throws UnknownFieldException if no {@link SimpleField} corresponding to {@code storageId} exists
1361     * @throws UnknownFieldException if {@code path} contains a storage ID that does not correspond to a {@link ReferenceField}
1362     * @throws IllegalArgumentException if {@code storageId} refers to a sub-field of a complex field
1363     * @throws IllegalArgumentException if {@code path} or {@code listener} is null
1364     * @throws StaleTransactionException if this transaction is no longer usable
1365     */

1367        this.validateChangeListener(SimpleField.class, storageId, path, listener);
1368        this.getMonitorsForField(storageId, true).add(new FieldMonitor(storageId, path, listener));
1369    }

1484    /**
1485     * Perform some action and, when entirely done (including re-entrant invocation), issue pending notifications to monitors.
1486     *
1487     * @param id object containing the mutated field; will be validated
1488     * @param mutation change to apply
1489     * @throws ReadOnlyTransactionException if this transaction has been {@linkplain #setReadOnly set read-only}
1490     * @throws StaleTransactionException if this transaction is no longer usable
1491     * @throws IllegalArgumentException if {@code id} is null
1492     */
1493    synchronized <V> V mutateAndNotify(ObjId id, Mutation<V> mutation) {
1494
1495        // Check that this transaction is still valid and the given object still exists.
1496        if (id == null)
1497            throw new IllegalArgumentException("null id");
1498        if (this.stale)
1499            throw new StaleTransactionException(this);
1500        if (this.readOnly)
1501            throw new ReadOnlyTransactionException(this);
1502        if (this.kvt.get(id.getBytes()) == null)
1503            throw new DeletedObjectException(id);
1504
1505        // If re-entrant invocation, we're already set up
1506        if (this.pendingNotifications.get() != null)
1507            return mutation.mutate();
1508
1509        // Set up pending report list, perform mutation, and then issue reports
1510        this.pendingNotifications.set(new TreeMap<Integer, ArrayList<FieldChangeNotifier>>());
1511        try {
1512            return mutation.mutate();
1513        } finally {
1514            try {
1515                final TreeMap<Integer, ArrayList<FieldChangeNotifier>> pendingNotificationMap = this.pendingNotifications.get();
1516                while (!pendingNotificationMap.isEmpty()) {
1517
1518                    // Get the next field with pending notifications
1519                    final Map.Entry<Integer, ArrayList<FieldChangeNotifier>> entry = pendingNotificationMap.pollFirstEntry();
1520                    final int storageId = entry.getKey();
1521
1522                    // For all pending notifications, back-track references and notify all field monitors for the field
1523                    for (FieldChangeNotifier notifier : entry.getValue()) {
1524                        assert notifier.getStorageId() == storageId;
1525                        final ArrayList<FieldMonitor> monitorList = new ArrayList<>(this.getMonitorsForField(storageId, false));
1526                        if (!monitorList.isEmpty())
1527                            this.notifyFieldMonitors(notifier, NavigableSets.singleton(notifier.getId()), monitorList, 0);
1528                    }
1529                }
1530            } finally {
1531                this.pendingNotifications.remove();
1532            }
1533        }
1534    }

1536    // Recursively back-track references along monitor paths and notify monitors when we reach the end (i.e., beginning)
1537    private void notifyFieldMonitors(FieldChangeNotifier notifier,
1538      NavigableSet<ObjId> objects, ArrayList<FieldMonitor> monitorList, int step) {
1539
1540        // Find the monitors for whom we have completed all the steps in their (inverse) path,
1541        // and group the remaining monitors by their next inverted reference path step.
1542        final HashMap<Integer, ArrayList<FieldMonitor>> remainingMonitorsMap = new HashMap<>();
1543        for (FieldMonitor monitor : monitorList) {
1544
1545            // Issue notification callback if we have back-tracked through the whole path
1546            if (monitor.path.length == step) {
1547                notifier.notify(this, monitor.listener, monitor.path, objects);
1548                continue;
1549            }
1550
1551            // Sort the unfinished monitors by their next (i.e., previous) reference field
1552            final int storageId = monitor.path[monitor.path.length - step - 1];
1553            ArrayList<FieldMonitor> remainingMonitors = remainingMonitorsMap.get(storageId);
1554            if (remainingMonitors == null) {
1555                remainingMonitors = new ArrayList<FieldMonitor>();
1556                remainingMonitorsMap.put(storageId, remainingMonitors);
1557            }
1558            remainingMonitors.add(monitor);
1559        }
1560
1561        // Invert references for each group of remaining monitors and recurse
1562        for (Map.Entry<Integer, ArrayList<FieldMonitor>> entry : remainingMonitorsMap.entrySet()) {
1563            final int storageId = entry.getKey();
1564
1565            // Gather all objects that refer to any object in our current "objects" set
1566            final ArrayList<NavigableSet<ObjId>> refsList = new ArrayList<>();
1567            for (ObjId object : objects) {
1568                final NavigableSet<ObjId> refs = this.queryIndex(storageId, FieldType.REFERENCE, FieldType.REFERENCE).get(object);
1569                if (refs != null)
1570                    refsList.add(refs);
1571            }
1572
1573            // Recurse on the union of those objects
1574            if (!refsList.isEmpty())
1575                this.notifyFieldMonitors(notifier, NavigableSets.union(refsList), entry.getValue(), step + 1);
1576        }
1577    }

1581    /**
1582     * Find all objects that refer to any object in the given target set through the specified path of references.
1583     *
1584     * @param path path of one or more reference fields (represented by storage IDs) through which to reach the target objects
1585     * @param targetObjects target objects
1586     * @return set of objects that refer to the {@code targetObjects} via {@code path}
1587     * @throws UnknownFieldException if {@code path} contains a storage ID that does not correspond to a {@link ReferenceField}
1588     * @throws IllegalArgumentException if {@code targetObjects} or {@code path} is null
1589     * @throws IllegalArgumentException if {@code path} is empty
1590     * @throws StaleTransactionException if this transaction is no longer usable
1591     */
1592    public NavigableSet<ObjId> invertReferencePath(int[] path, Iterable<ObjId> targetObjects) {
1593
1594        // Sanity check
1595        if (targetObjects == null)
1596            throw new IllegalArgumentException("null targetObjects");
1597        if (path == null)
1598            throw new IllegalArgumentException("null path");
1599        if (path.length == 0)
1600            throw new IllegalArgumentException("empty path");
1601
1602        // Verify all fields in the path are reference fields
1603        for (int pathStorageId : path)
1604            this.schema.verifyStorageInfo(pathStorageId, ReferenceFieldStorageInfo.class);
1605
1606        // Invert references
1607        NavigableSet<ObjId> result = null;
1608        for (int i = path.length - 1; i >= 0; i--) {
1609            final int storageId = path[i];
1610
1611            // Gather all objects that refer to any object in our current target objects set
1612            final ArrayList<NavigableSet<ObjId>> refsList = new ArrayList<>();
1613            for (ObjId id : targetObjects) {
1614                final NavigableSet<ObjId> refs = this.queryIndex(storageId, FieldType.REFERENCE, FieldType.REFERENCE).get(id);
1615                if (refs != null)
1616                    refsList.add(refs);
1617            }
1618            if (refsList.isEmpty())
1619                return NavigableSets.empty();
1620
1621            // Recurse on the union of those objects
1622            targetObjects = result = NavigableSets.union(refsList);
1623        }
1624
1625        // Done
1626        return result;
1627    }

1882    /**
1883     * Callback interface for notification of transaction completion events.
1884     *
1885     * <p>
1886     * Modeled after Spring's {@link org.springframework.transaction.support.TransactionSynchronization} interface.
1887     * </p>
1888     */
1889    public interface Callback {
1890
1891        /**
1892         * Invoked before transaction commit (and before any {@link #beforeCompletion}).
1893         * This method is invoked when a transaction is intended to be committed; it may however still be rolled back.
1894         *
1895         * <p>
1896         * Any exceptions thrown will result in a transaction rollback and be propagated to the caller.
1897         * </p>
1898         *
1899         * @param readOnly true if the transaction {@linkplain Transaction#isReadOnly is marked read-only}
1900         */
1901        void beforeCommit(boolean readOnly);
1902
1903        /**
1904         * Invoked before transaction completion in any case (but after any {@link #beforeCommit}).
1905         * This method is invoked whether the transaction is going to be committed or rolled back,
1906         * and is invoked even if {@link #beforeCommit} throws an exception.
1907         * Typically used to clean up resources before transaction completion.
1908         *
1909         * <p>
1910         * Any exceptions thrown will be logged but will <b>not</b> propagate to the caller.
1911         * </p>
1912         */
1913        void beforeCompletion();
1914
1915        /**
1916         * Invoked after successful transaction commit (and before any {@link #afterCompletion}).
1917         *
1918         * <p>
1919         * Any exceptions thrown will propagate to the caller.
1920         * </p>
1921         */
1922        void afterCommit();
1923
1924        /**
1925         * Invoked after transaction completion (but after any {@link #afterCommit}).
1926         * This method is invoked in any case, whether the transaction was committed or rolled back.
1927         * Typically used to clean up resources after transaction completion.
1928         *
1929         * <p>
1930         * Any exceptions thrown will be logged but will <b>not</b> propagate to the caller.
1931         * </p>
1932         *
1933         * @param committed true if transaction was commited, false if transaction was rolled back
1934         */
1935        void afterCompletion(boolean committed);
1936    }
