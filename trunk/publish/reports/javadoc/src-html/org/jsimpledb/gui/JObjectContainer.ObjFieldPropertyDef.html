054/**
055 * General purpose support superclass for Vaadin {@link com.vaadin.data.Container}s backed by {@link JSimpleDB} database objects.
056 * Automatically creates properties for object ID, database type, version, and fields, as well as any custom properties
057 * defined by {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty}-annotated methods in Java model classes,
058 * and handles {@link Change} updates of backing objects or their related objects.
059 *
060 * <p>
061 * Instances are configured with a <b>type</b>, which can be any Java type (including interface types). The container
062 * will then be restricted to database objects that are instances of the configured type. The type may be null, in which
063 * case there is no restriction. The subclass method {@linkplain #queryForObjects queryForObjects()} determines which
064 * objects are actually loaded into the container. The items in the container are backed by in-memory copies of the
065 * corresponding database objects that live inside a {@link org.jsimpledb.SnapshotJTransaction}.
066 * </p>
067 *
068 * <p><b>Container Properties</b></p>
069 *
070 * <p>
071 * Instances will have the following container properties:
072 * <ul>
073 *  <li>{@link #OBJECT_ID_PROPERTY}: Object {@link ObjId}</li>
074 *  <li>{@link #TYPE_PROPERTY}: Object type name (JSimpleDB type name, not Java type name, though these are typically the same)</li>
075 *  <li>{@link #VERSION_PROPERTY}: Object schema version</li>
076 *  <li>{@link #REFERENCE_LABEL_PROPERTY}: Object <b>reference label</b>, which is a short description identifying the
077 *      object. Reference labels are used to provide "names" for objects that are more meaningful than object ID's
078 *      and are used as such in other {@link JSimpleDB} GUI classes. To customize the reference label for a Java model class,
079 *      annotate a method with {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty}{@code (}{@link
080 *      JObjectContainer#REFERENCE_LABEL_PROPERTY REFERENCE_LABEL_PROPERTY}{@code )};
081 *      otherwise, the value of this property will be the same as {@link #OBJECT_ID_PROPERTY}.</i>
082 *  <li>A property for every {@link JSimpleDB} field that is common to all object types that sub-type
083 *      this instance's configured type. The property's ID is the field name; its value is as follows:
084 *      <ul>
085 *          <li>Simple fields will show their string values</li>
086 *          <li>Reference fields show the {@link #REFERENCE_LABEL_PROPERTY} of the referred-to object, or "Null"
087 *              if the reference is null</li>
088 *          <li>Set, list, and map fields show the first few entries in the collection</li>
089 *      </ul>
090 *  </li>
091 *  <li>A property for each {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty}-annotated method
092 *      in the specified <b>type</b>. These properties will add to (or override) the properties listed above.
093 * </ul>
094 * </p>
095 *
096 * <p><b>Loading</b></p>
097 *
098 * <p>
099 * Instances may be (re)loaded at any time by invoking {@link #reload}. This causes the container
100 * to query for {@link JObject}s within a new {@link JTransaction} via the subclass-provided method
101 * {@link #queryForObjects queryForObjects()}, which returns an {@link Iterable Iterable<JObject>}. This container
102 * copies the resulting database objects into the container's
103 * in-memory {@link org.jsimpledb.SnapshotJTransaction}. The latter effectively serves as the cache for the container,
104 * so that database transactions may be short-lived and are only required when (re)loading. Because the container contents
105 * are in memory, with large data sets this container should only be loaded with one "page" of objects at a time (a "page"
106 * typically being only what a human would be willing to scroll through at one time). It is up to
107 * {@link #queryForObjects queryForObjects()} to determine what and how many objects are returned, their sort order, etc.
108 * (if {@link #queryForObjects queryForObjects()} returns any objects that are not instances of the configured type,
109 * they are ignored). In any case, the maximum number of objects that will be loaded is limited by a
110 * {@linkplain #setMaxObjects configurable} maximum.
111 * </p>
112 *
113 * <p>
114 * When {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty}-annotated methods are present on Java
115 * model classes, these property values may derive from other objects related to the backing instance. In order for these
116 * methods to work on the in-memory instance, the related objects must be copied into memory as well. The
117 * {@link #getDependencies getDependencies()} method allows the subclass to specify what other database objects
118 * some Vaadin property depends on.
119 * </p>
120 *
121 * <p><b>Updating Items</b></p>
122 *
123 * <p>
124 * Instances support updating individual items without requiring a complete container reload by invoking
125 * {@link #handleChange handleChange()}. In a typical design pattern, {@link org.jsimpledb.change.Change} objects originate
126 * from {@link org.jsimpledb.annotation.OnChange &#64;OnChange} methods and are broadcast (after copying into memory using
127 * a {@link org.jsimpledb.change.ChangeCopier}) to listeners such as this container by a transaction synchronization callback
128 * (see {@link org.jsimpledb.core.Transaction#addCallback Transaction.addCallback()}).
129 * </p>
130 *
131 * <p>
132 * Moreover, this container keeps track of which related objects are associated with each backing object (according to
133 * {@link #getDependencies getDependencies()}). {@link Change} events that affect a related object automatically cause
134 * the corresponding backing object(s) to update.
135 * </p>
136 *
137 * <p>
138 * This class responds to the creation or deletion of a database object matching this container's configured type
139 * by reloading the container; however, this is a conservative approach, as not necessarily every newly created (or
140 * deleted) object should be (or was) present in the container in the first place. Subclasses may wish to optimize this
141 * behavior by overriding {@link #handleObjectCreate handleObjectCreate()} and/or {@link #handleObjectDelete handleObjectDelete()}.
142 * </p>
143 *
144 * <p><b>{@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty} Limitations</b></p>
145 *
146 * <p>
147 * The use of {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty} methods has certain implications.
148 * First, if the method reads any of the object's field(s) via their Java getter methods (as would normally be expected),
149 * this will trigger a schema upgrade of the object if needed; however, this schema upgrade will occur in the
150 * container's in-memory {@link org.jsimpledb.SnapshotJTransaction} rather than in a real database transaction, so the
151 * {@link #VERSION_PROPERTY} will return a different schema version from what's in the database. The automatic schema
152 * upgrade can be avoided if desired by reading the field using the appropriate {@link JTransaction} field access method
153 * (e.g., {@link JTransaction#readSimpleField JTransaction.readSimpleField()}) and being prepared to handle a
154 * {@link org.jsimpledb.core.UnknownFieldException} if/when the object has an older schema version that does not contain
155 * the requested field.
156 * </p>
157 *
158 * <p>
159 * Secondly, because the values of reference fields (including complex sub-fields) are displayed using reference labels,
160 * and these are typically derived from the referenced object's fields, those indirectly referenced objects need to be
161 * copied into the container's {@link org.jsimpledb.SnapshotJTransaction} as well. The easiest way to ensure these indirectly
162 * referenced objects are copied is by overriding {@link #getDependencies getDependencies()} as described above. Alternately,
163 * the subclass may override {@link #copyOut copyOut()} to take complete control of copying objects into memory.
164 * </p>
165 */

293    /**
294     * Apply the given change to this container, if applicable.
295     *
296     * <p>
297     * The changed object {@linkplain Change#getObject referred to} by {@code change} must be accessible.
298     * Unless this method is invoked from within the transaction in which the change occurred, this means that
299     * {@code change} must have been be copied out of its transaction, e.g, using a {@link org.jsimpledb.change.ChangeCopier}.
300     * </p>
301     *
302     * <p>
303     * The implementation in {@link JObjectContainer} delegates to {@link #handleObjectCreate handleObjectCreate()},
304     * {@link #handleObjectDelete handleObjectDelete()}, or {@link #handleFieldChange handleFieldChange()} as appropriate.
305     * </p>
306     *
307     * @throws IllegalArgumentException if {@code change} is null
308     * @throws org.jsimpledb.core.StaleTransactionException if {@code change} refers to a {@link JObject}
309     *  without an associated tranasction.
310     */
311    public <T> void handleChange(Change<T> change) {
312
313        // Sanity check
314        if (change == null)
315            throw new IllegalArgumentException("null change");
316
317        // Determine whether the change concerns us
318        if (this.type != null && !this.type.isInstance(change.getObject()))
319            return;
320
321        // Apply change
322        change.visit(new ChangeAdapter<Void>() {
323
324            @Override
325            public <T> Void caseObjectDelete(ObjectDelete<T> change) {
326                JObjectContainer.this.handleObjectDelete(change);
327                return null;
328            }
329
330            @Override
331            public <T> Void caseObjectCreate(ObjectCreate<T> change) {
332                JObjectContainer.this.handleObjectCreate(change);
333                return null;
334            }
335
336            @Override
337            protected <T> Void caseFieldChange(FieldChange<T> change) {
338                JObjectContainer.this.handleFieldChange(change);
339                return null;
340            }
341        });
342    }

366    /**
367     * Handle a {@link FieldChange} event.
368     *
369     * <p>
370     * The implementation in {@link JObjectContainer} triggers value change events for all affected item(s) in this container.
371     * Affected items are those whose backing object is either the changed object, or has the changed object as a related object.
372     * </p>
373     */
374    protected <T> void handleFieldChange(FieldChange<T> change) {
375
376        // Update the matching backing object, if any
377        final JObject target = change.getJObject();
378        final SimpleItem<JObject> item = (SimpleItem<JObject>)this.getItem(target.getObjId());
379        if (item != null) {
380            target.copyTo(item.getObject().getTransaction(), null, new ObjIdSet());
381            item.fireValueChange();
382        }
383
384        // Update any other affected objects
385        final ObjIdSet affectedIds = dependenciesMap.get(target.getObjId());
386        if (affectedIds != null) {
387            for (ObjId affectedId : affectedIds) {
388                final SimpleItem<JObject> affectedItem = (SimpleItem<JObject>)this.getItem(affectedId);
389                if (affectedItem != null)
390                    affectedItem.fireValueChange();
391            }
392        }
393    }

395    // This method runs within a transactin
396    private void doReloadInTransaction() {
397
398        // Get objects from subclass
399        Iterable<? extends JObject> jobjs = this.queryForObjects();
400
401        // Filter out any instances of the wrong type
402        if (JObjectContainer.this.type != null) {
403            jobjs = Iterables.filter(jobjs, new Predicate<JObject>() {
404                @Override
405                public boolean apply(JObject jobj) {
406                    return JObjectContainer.this.type.isInstance(jobj);
407                }
408            });
409        }
410
411        // Filter out any duplicate objects using ObjId set
412        final ObjIdSet seen = new ObjIdSet();
413
414        // Copy database objects out of the database transaction into my in-memory transaction as we load them
415        jobjs = Iterables.transform(jobjs, new Function<JObject, JObject>() {
416            @Override
417            public JObject apply(JObject jobj) {
418                return JObjectContainer.this.copyOut(jobj, seen);
419            }
420        });
421
422        // Limit the total number of objects in the container
423        jobjs = Iterables.limit(jobjs, this.maxObjects);
424
425        // Now actually load them
426        this.dependenciesMap.clear();
427        this.load(jobjs);
428    }

430    /**
431     * Copy the given database object, and any related objects needed by any
432     * {@link org.dellroad.stuff.vaadin7.ProvidesProperty &#64;ProvidesProperty}-annotated methods,
433     * into the current {@link org.jsimpledb.SnapshotJTransaction}.
434     *
435     * <p>
436     * The implementation in {@link JObjectContainer} copies {@code jobj}, and all of {@code jobj}'s related objects returned
437     * by {@link #getDependencies getDependencies()}, via {@link JTransaction#copyTo(JTransaction, ObjIdSet, Iterable)}.
438     * </p>
439     *
440     * @param target the object to copy, or null (ignored)
441     * @param seen object ID's already copied
442     * @return the copy of {@code target} in the current {@link org.jsimpledb.SnapshotJTransaction},
443     *  or null if {@code target} is null
444     */
445    protected JObject copyOut(JObject target, ObjIdSet seen) {
446
447        // Ignore null
448        if (target == null)
449            return null;
450
451        // Copy out object
452        final ObjId targetId = target.getObjId();
453        final JTransaction jtx = JTransaction.getCurrent();
454        final SnapshotJTransaction sjtx = jtx.getSnapshotTransaction();
455        final JObject copy = target.copyTo(sjtx, null, seen);
456
457        // Copy out (and track) related objects
458        Iterable<? extends JObject> dependencies = this.getDependencies(target);
459        if (dependencies != null) {
460            dependencies = Iterables.filter(dependencies, JObject.class);                       // filter out nulls
461            jtx.copyTo(sjtx, seen, dependencies);
462            for (JObject jobj : dependencies) {
463                final ObjId id = jobj.getObjId();
464                ObjIdSet affectedIds = this.dependenciesMap.get(id);
465                if (affectedIds == null) {
466                    affectedIds = new ObjIdSet();
467                    this.dependenciesMap.put(id, affectedIds);
468                }
469                affectedIds.add(targetId);
470            }
471        }
472
473        // Done
474        return copy;
475    }

518    private Collection<PropertyDef<?>> buildPropertyDefs() {
519        final PropertyDefHolder pdefs = new PropertyDefHolder();
520
521        // Add properties shared by all JObjects
522        pdefs.setPropertyDef(this.refLabelPropertyDef);
523        pdefs.setPropertyDef(this.objIdPropertyDef);
524        pdefs.setPropertyDef(this.objTypePropertyDef);
525        pdefs.setPropertyDef(this.objVersionPropertyDef);
526
527        // Add properties for all fields common to all sub-types of our configured type
528        final SortedMap<Integer, JField> jfields = Util.getCommonJFields(this.jdb.getJClasses(this.type));
529        if (jfields != null) {
530            for (JField jfield : jfields.values())
531                pdefs.setPropertyDef(new ObjFieldPropertyDef(jfield.getStorageId(), jfield.getName()));
532        }
533
534        // Apply any @ProvidesProperty-annotated method properties, possibly overridding jfields
535        if (this.propertyScanner != null) {
536            for (PropertyDef<?> propertyDef : this.propertyScanner.getPropertyDefs())
537                pdefs.setPropertyDef(propertyDef);
538        }
539
540        // Done
541        return pdefs.values();
542    }

549        if (propertyDef instanceof ObjPropertyDef)
550            return (V)((ObjPropertyDef<?>)propertyDef).extract(jobj);
551        if (this.propertyScanner == null)

556    @SuppressWarnings("unchecked")
557    private static <V> V extractProperty(PropertyExtractor<?> propertyExtractor, PropertyDef<V> propertyDef, JObject jobj) {
558        try {
559            return ((PropertyExtractor<JObject>)propertyExtractor).getPropertyValue(jobj, propertyDef);
560        } catch (DeletedObjectException e) {
561            try {
562                return propertyDef.getType().cast(new SizedLabel("<i>Missing</i>", ContentMode.HTML));
563            } catch (ClassCastException e2) {
564                try {
565                    return propertyDef.getType().cast("(Missing)");
566                } catch (ClassCastException e3) {
567                    return null;
568                }
569            }
570        }
571    }

642     * Implements the {@link JObjectContainer#REFERENCE_LABEL_PROPERTY} property.
643     */
644    public static class RefLabelPropertyDef extends ObjPropertyDef<Component> {
645
646        public RefLabelPropertyDef() {
647            super(REFERENCE_LABEL_PROPERTY, Component.class);
648        }
649
650        @Override
651        public Component extract(JObject jobj) {
652            final ReferenceMethodInfoCache.PropertyInfo<?> propertyInfo
653              = ReferenceMethodInfoCache.getInstance().getReferenceMethodInfo(jobj.getClass());
654            if (propertyInfo == ReferenceMethodInfoCache.NOT_FOUND)
655                return new ObjIdPropertyDef().extract(jobj);
656            final Object value = JObjectContainer.extractProperty(
657              propertyInfo.getPropertyExtractor(), propertyInfo.getPropertyDef(), jobj);
658            if (value instanceof Component)
659                return (Component)value;
660            return new SizedLabel(String.valueOf(value));
661        }
662    }

666    /**
667     * Implements a property reflecting the value of a {@link JSimpleDB} field.
668     */
669    public class ObjFieldPropertyDef extends ObjPropertyDef<Component> {
670
671        private static final int MAX_ITEMS = 3;
672
673        private final int storageId;
674
675        public ObjFieldPropertyDef(int storageId, String name) {
676            super(name, Component.class);
677            this.storageId = storageId;
678        }
679
680        @Override
681        public Component extract(final JObject jobj) {
682            final JField jfield = JObjectContainer.this.jdb.getJClass(jobj.getObjId()).getJField(this.storageId, JField.class);
683            try {
684                return jfield.visit(new JFieldSwitchAdapter<Component>() {
685
686                    @Override
687                    public Component caseJSimpleField(JSimpleField field) {
688                        return ObjFieldPropertyDef.this.handleValue(field.getValue(jobj));
689                    }
690
691                    @Override
692                    public Component caseJCounterField(JCounterField field) {
693                        return ObjFieldPropertyDef.this.handleValue(field.getValue(jobj).get());
694                    }
695
696                    @Override
697                    protected Component caseJCollectionField(JCollectionField field) {
698                        return ObjFieldPropertyDef.this.handleCollectionField(field.getValue(jobj));
699                    }
700
701                    @Override
702                    public Component caseJMapField(JMapField field) {
703                        return ObjFieldPropertyDef.this.handleMultiple(Iterables.transform(field.getValue(jobj).entrySet(),
704                          new Function<Map.Entry<?, ?>, Component>() {
705                            @Override
706                            public Component apply(Map.Entry<?, ?> entry) {
707                                final HorizontalLayout layout = new HorizontalLayout();
708                                layout.setMargin(false);
709                                layout.setSpacing(false);
710                                layout.addComponent(ObjFieldPropertyDef.this.handleValue(entry.getKey()));
711                                layout.addComponent(new SizedLabel(" \u21d2 "));        // RIGHTWARDS DOUBLE ARROW
712                                layout.addComponent(ObjFieldPropertyDef.this.handleValue(entry.getValue()));
713                                return layout;
714                            }
715                        }));
716                    }
717                });
718            } catch (UnknownFieldException e) {
719                return new SizedLabel("<i>NA</i>", ContentMode.HTML);
720            }
721        }
722
723        private Component handleCollectionField(Collection<?> col) {
724            return this.handleMultiple(Iterables.transform(col, new Function<Object, Component>() {
725                @Override
726                public Component apply(Object item) {
727                    return ObjFieldPropertyDef.this.handleValue(item);
728                }
729            }));
730        }
731
732        private Component handleMultiple(Iterable<Component> components) {
733            final HorizontalLayout layout = new HorizontalLayout();
734            layout.setMargin(false);
735            layout.setSpacing(false);
736            int count = 0;
737            for (Component component : components) {
738                if (count >= MAX_ITEMS) {
739                    layout.addComponent(new SizedLabel("..."));
740                    break;
741                }
742                if (count > 0)
743                    layout.addComponent(new SizedLabel(",&#160;", ContentMode.HTML));
744                layout.addComponent(component);
745                count++;
746            }
747            return layout;
748        }
749
750        @SuppressWarnings("unchecked")
751        private <T> Component handleValue(Object value) {
752            if (value == null)
753                return new SizedLabel("<i>Null</i>", ContentMode.HTML);
754            if (value instanceof JObject)
755                return new RefLabelPropertyDef().extract((JObject)value);
756            return new SizedLabel(String.valueOf(value));
757        }
758    }
