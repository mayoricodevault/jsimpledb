****** DATABASE META-DATA *******

Format Version

    +------+------+-----------------+    +-------------------+
    | 0x00 | 0x00 |   'JSimpleDB'   |    |   Format Version  |
    +------+------+-----------------+    +-------------------+

Recorded Schemas

    +------+------+-------------------+---------------------------+
    | 0x00 | 0x01 |   Version Number  |   Compressed Schema XML   |
    +------+------+-------------------+---------------------------+

Object Version Index

    +------+------+-------------------+---------------+
    | 0x00 | 0x80 |   Version Number  |     ObjId     |
    +------+------+-------------------+---------------+

****** OBJECTS *******

    +------------+   +-------------------+----------------+-----------------+
    |  Object ID |   | Meta-Data Version | Object Version | Delete Notified |
    +------------+   +-------------------+----------------+-----------------+
