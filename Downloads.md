```xml

There is also a traditional distribution ZIP file that contains executable JARs for the command line interface (CLI) and graphical user interface (GUI), as well as a demonstration database you can play with. The distribution ZIP file, as well as the Maven artifacts mentioned above, are available here:

  * [jsimpledb-1.1.838.zip](https://s3.amazonaws.com/archie-public/jsimpledb/jsimpledb-1.1.838.zip) - Distribution ZIP file (includes everything below plus demo database)
  * [jsimpledb-1.1.838.jar](https://s3.amazonaws.com/archie-public/jsimpledb/jsimpledb-1.1.838.jar) - Main JAR file
  * [jsimpledb-cli-1.1.838.jar](https://s3.amazonaws.com/archie-public/jsimpledb/jsimpledb-cli-1.1.838.jar) - CLI classes
  * [jsimpledb-gui-1.1.838.jar](https://s3.amazonaws.com/archie-public/jsimpledb/jsimpledb-gui-1.1.838.jar) - GUI classes
  * [jsimpledb-spring-1.1.838.jar](https://s3.amazonaws.com/archie-public/jsimpledb/jsimpledb-spring-1.1.838.jar) - Spring Framework support classes
  * [jsimpledb-javadoc-1.1.838.jar](https://s3.amazonaws.com/archie-public/jsimpledb/jsimpledb-javadoc-1.1.838.jar) - API Javadocs
  * [jsimpledb-sources-1.1.838.jar](https://s3.amazonaws.com/archie-public/jsimpledb/jsimpledb-sources-1.1.838.jar) - Library source files

Changes are noted [here](http://jsimpledb.googlecode.com/svn/trunk/CHANGES.txt).
